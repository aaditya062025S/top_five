## About This Project
Top Five is a social media platform where users create posts listing their 
top five favorite items for a specific category (e.g., top five movies, 
top five musicians, top five sports, etc.).

## How to Run Locally
1. Install dependencies:
pip install -r requirements.txt

2. Start the app:
python app.py

3. Open your browser and go to: http://127.0.0.1:5000/

## CI/CD
This project uses GitHub Actions for continuous integration.
The workflow automatically runs on every push to verify the build.
